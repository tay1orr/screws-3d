# 이스터에그·진행도 기반 업데이트

## 목적

현재 한 레벨 게임을 다음 맵 제작이 가능한 구조로 확장하고, 메뉴에 개인적인 제작자 이스터에그를 추가한다.

## 화면 개선

- 오두막이 화면에서 더 크고 위쪽에 보이도록 카메라 거리와 세로 중심 조정
- 확대 시 섬이 잘리지 않도록 섬 반지름 동시 축소
- 수집 UI 뒤를 가리던 구름을 화면 외곽으로 이동
- 하단 조작 안내에 반투명 배경과 대비 강화
- 시작 화면 종료 후 접근성 트리에서도 제거

## 제작자 이스터에그

- 일시정지 메뉴에 `제작자: taylorr ♥` 버튼 추가
- 클릭하면 게임 스타일의 날짜 입력 모달 표시
- 숫자 여덟 자리만 허용하고 값은 저장하거나 외부로 전송하지 않음
- 잘못된 날짜는 정답을 노출하지 않고 카드 흔들림과 안내 문구 표시
- 지정 날짜 성공 시 다음 효과 재생
  - 최대 240개 하트 파티클
  - 네 차례 충격파
  - 여러 화음으로 구성한 전용 WebAudio 효과음
  - 지원 기기의 짧은 진동 패턴
  - 모션 감소 설정에서는 파티클 수와 지속시간 감소

## 레벨 확장 기반

- 현재 레벨에 ID, 이름, 월드, 난이도, 설명 메타데이터 추가
- `LEVEL_SUMMARY`로 레벨 선택 화면에서 사용할 요약 데이터 자동 생성
- localStorage 기반 완료 상태와 다음 레벨 잠금 해제 저장
- 마지막 레벨 완료 시 ‘다음 레벨’ 대신 ‘다시 플레이’ 표시

## 검증 결과

- JavaScript 문법 검사 통과
- 기존 게임 규칙 테스트 12/12 통과
- 진행도 저장 테스트 3/3 통과
- `git diff --check` 통과
- 정적 서버에서 HTML, CSS, 핵심 모듈 HTTP 200 확인
