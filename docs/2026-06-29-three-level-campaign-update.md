# 3레벨 캠페인 업데이트

## 결과

한 개의 오두막만 반복하던 구조를 순차 해제되는 3레벨 캠페인으로 확장했다.

## 레벨 구성

1. `첫 나사판` — 18개 나사, 3색, 난이도 1
   - 모든 나사가 시작 시점부터 보임
   - 상자 직행과 임시 보관함을 짧게 학습
2. `하늘빛 풍차` — 48개 나사, 4색, 난이도 5
   - 네 개의 풍차 날개와 중심 허브
   - 앞쪽 부품을 제거해 문과 벽 나사를 노출
   - 측면·후면 확인을 위한 회전 학습
3. `다층 오두막` — 84개 나사, 6색, 난이도 10
   - 기존 오두막을 보스 레벨로 재배치
   - 문·창·지붕·내부 바닥의 다층 차폐

## 레벨 선택 화면

- 시작 화면과 일시정지 메뉴에서 접근
- 레벨 번호, 이름, 난이도, 나사 수 표시
- 잠긴 레벨과 완료 레벨을 서로 다른 시각 상태로 표시
- 레벨 완료 후 다음 카드 자동 해제
- 마지막으로 열린 레벨을 다음 접속 시작점으로 사용

## 진행도 저장

- 캠페인 변경과 기존 단일 레벨 저장 충돌을 막기 위해 저장 스키마를 v2로 변경
- 완료 레벨과 최대 해제 레벨만 브라우저 localStorage에 기록
- 개인정보나 날짜 이스터에그 입력값은 저장하지 않음

## 검증

- 새 캠페인 레벨 정적 데이터 검사 통과
- 튜토리얼 18개, 풍차 48개 나사 수 확인
- 모든 색상 나사 수가 3의 배수
- 상자 큐 용량과 실제 나사 수 일치
- 규칙 엔진 기반 이론 풀이 완료
- 기존 규칙 테스트 12/12 통과
- 진행도 저장 테스트 3/3 통과
- 캠페인 테스트 2/2 통과

## 배포 후 확인 필요

- 360×800, 390×844, 1280×720 레벨 카드 배치
- 풍차 날개 방향과 나사 차폐 상태
- 각 레벨 완료 후 실제 잠금 해제 흐름
- 이스터에그 하트 파티와 레벨 패널 z-index 충돌 여부
